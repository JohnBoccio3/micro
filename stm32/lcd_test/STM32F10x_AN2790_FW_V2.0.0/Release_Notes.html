What's New in AN2790 "TFT LCD interfacing with the high-density STM32F10xxx FSMC" V2.0.0

The AN2790 "TFT LCD interfacing with the high-density STM32F10xxx FSMC" V2.0.0 is an update of V1.0.1 that makes the application note compliant with the Cortex™ microcontroller software interface standard (CMSIS).

What is AN2790 "TFT LCD interfacing with the high-density STM32F10xxx FSMC"?

The AN2790 "TFT LCD interfacing with the high-density STM32F10xxx FSMC" describes a firmware TFT LCD interfacing with the High Density STM32F10xxx  FSMC peripheral. The main purpose of this firmware package is to provide resources  facilitating the development of an application using the LCD on FSMC peripheral.
The firmware interface is composed of library source files developed in order to support LCD features, an application demo is also provided.

The AN2790 "TFT LCD interfacing with the high-density STM32F10xxx FSMC" is full CMSIS compliant.
